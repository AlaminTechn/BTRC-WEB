## Overview

This document defines a **5-tab functional grouping** for the Executive Dashboard, organizing 13 components by decision-making function rather than topic area.

### Design Principles

- **Decision-Aligned**: Each tab answers a specific executive question
- **Data Source Coherence**: Components within tabs share primary data sources
- **Reduced Cognitive Load**: 5 functional groups vs 6 topic sections
- **Within-Country Focus**: All comparisons are internal (division/district level)

|-----------|-------------|

| Performance benchmarking | Division-level performance tiers (top/average/bottom performers); best-practice identification; improvement recommendations |
